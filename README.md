# Backend template TS v.2022

### 💻 nodeJS backend template

- express
- typescript
- cors
- dotenv
- ts-node
- nodemon
- git-commit-msg-linter
- husky
- lint-staged

##### ⚙️ For use you must run:

```
npm install
```

##### ⚙️ For development

```
npm run dev
```

##### ⚙️ Build for production

It Will create a 'dist' folder

```
npm run build
```

##### ⚙️ Run on production

```
npm run start
```

##### ⚙️ Environment variables

Check the '.env.example' file
